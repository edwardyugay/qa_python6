# Sprint_6

Проект для автоматизированного тестирования сервиса «Яндекс.Самокат».
В проекте реализованы автотесты с использованием Selenium, Page Object Model, pytest, Allure и Faker для генерации динамических тестовых данных.

## Структура проекта

- **conftest.py** – Фикстуры для настройки браузера.
- **helper.py** – Дополнительные утилиты (общие функции, если понадобятся).
- **urls.py** – Модуль с константами URL.
- **base_page.py** – Базовый класс для всех страниц, содержащий общие методы работы с driver.
- **test_data.py** – Генерация случайных тестовых данных с помощью Faker.
- **page_objects/** – Классы Page Object (MainPage, OrderPage), наследующие BasePage и использующие локаторы, вынесенные в атрибуты.
- **tests/** – Тестовые сценарии с параметризацией (FAQ, заказ).
- **requirements.txt** – Список зависимостей.
- **.gitignore** – Файлы, исключаемые из репозитория.
- **README.md** – Документация по проекту.
