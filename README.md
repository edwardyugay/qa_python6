# Sprint_6

Проект для автоматизированного тестирования сервиса «Яндекс.Самокат».
В проекте реализованы автотесты с использованием Selenium, Page Object Model, pytest, Allure и Faker для генерации случайных тестовых данных.

## Структура проекта

- **conftest.py** – Фикстуры для настройки браузера.
- **helper.py** – Вспомогательные функции (ожидание элементов, переключение окон).
- **test_data.py** – Генерация случайных тестовых данных с помощью Faker.
- **page_objects/** – Классы Page Object для главной страницы и страницы заказа.
- **tests/** – Тестовые сценарии с параметризацией.
- **requirements.txt** – Список зависимостей.
- **.gitignore** – Файлы, исключаемые из репозитория.
- **README.md** – Документация по проекту.
